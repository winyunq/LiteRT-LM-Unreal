Packaging and distributing large local models is a unique challenge. Learn how to manage the "last mile" of local AI deployment.

Distribution Strategies

LLM weight files (2GB - 8GB+) don't fit well into traditional .pak files due to memory mapping requirements.

External Delivery (Recommended)

Keep .bin and .tflite files in a subfolder like [GameDir]/Models/. Use the Non-UFS packaging mode to ensure the files remain raw on disk for mmap access.

On-Demand Download

Ship the game with a Base Agent and allow users to download higher-quality models (7B, 13B) via an in-game downloader.

VRAM Budgeting

When shipping to a wide range of hardware, you must define Quality Tiers.

Tier Recommended Model VRAM Target
Standard Gemma-2B (4-bit) 2GB - 3GB
Ultra Llama-3-8B (8-bit) 6GB - 8GB

Platform Notes

  • Windows: Ensure the application has read/write permissions for the external model folder.
  • Console (PS5/Xbox): Contact Winyunq for specialized LiteRT kernels optimized for console memory architectures.
