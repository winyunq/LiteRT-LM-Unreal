PHYSICAL LOGIC ATLAS

Stage: Bottom Layer (C-ABI)

Concept 01 / ABI Firewall

物理隔离：ABI 防火墙机制

物理契约： 虚幻侧仅持有 void*。底层所有 C++ 对象的生命周期、符号链接、内存布局均被 DLL 彻底遮蔽。
后果： 即使底层 Google SDK 版本升级，虚幻项目的二进制文件也无需重新编译。彻底解决了“符号冲突”这一大项目顽疾。
Concept 02 / KV Cache persistence

持久化：多会话 KV Cache 映射

物理映射： 每一个 Conversation 句柄在显存中独占一个 max_num_tokens 大小的物理分片。
瞬间切换： 切换对话对象只需在 RunInference 时传入不同的句柄。切换开销 < 1ms，因为记忆已经在显存中“静止”等待。
Concept 03 / Inference Heartbeat

执行驱动：推理心脏泵

核心动力契约： RunInference 仅仅是向 GPU 提交了计算工单。必须在后台线程中通过 WaitUntilDone 不断“泵送”引擎，计算结果才会通过回调吐回给 Game Thread。
