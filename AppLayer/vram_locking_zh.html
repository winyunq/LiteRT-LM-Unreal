01 / 具体操作步骤
Step 1: 物理环境探测

调用 GetAutoConfig() 获取当前机器的显存预算建议。严禁硬编码显存量，防止在低端显卡上触发 OOM。

Step 2: 异步封送

由于 LoadModel 会导致数秒的磁盘 IO 与显存寻址，必须通过 AsyncTask 将其从 GameThread 剥离。

Step 3: 状态监听

在加载成功前，业务层应挂起所有推理请求。建议在 UI 上展示 Loading 进度。

// 参考 FUmgMcpLiteRtLmAiProvider.cpp
void EnsureModelLoadedAsync(TFunction<void(bool)> OnComplete) {
    // 1. 配置
    FLiteRtLmConfig Config = FLiteRtLmUnrealApi::GetAutoConfig();
    Config.ModelPath = AbsPath;

    // 2. 异步执行
    AsyncTask(ENamedThreads::AnyBackgroundThreadNormalTask, [Config, OnComplete]() {
        bool bSuccess = FLiteRtLmUnrealApi::LoadModel(Config);
        
        // 3. 回调分发
        AsyncTask(ENamedThreads::GameThread, [bSuccess, OnComplete]() {
            if (OnComplete) OnComplete(bSuccess);
        });
    });
}
