物理 KV Cache 持久性

在 LiteRtLmUnrealApi.cpp 中，系统通过 SessionMsgCountMap 精确跟踪每个 Agent 已同步的消息索引。切换 Agent 就像在 GPU 显存中切换指针，实现了零开销的多 Agent 交互。

"这是将首字延迟 (TTFT) 从秒级降低到毫秒级的核心黑科技。"
