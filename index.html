让您的虚幻项目具备 本地 AI 之魂

LiteRT-LM Unreal 是专为虚幻引擎打造的高性能、轻量级本地大模型集成方案。仅需 $19，即可在 RTX 4060 上实现秒级 Agent 切换与流式推理。

快速集成示例 (C++)

在您的 Actor 中仅需几行代码即可唤醒 AI。我们独特的 Session 映射机制确保了 KV Cache 的极速命中。

// 1. 加载模型
FLiteRtLmConfig Config;
Config.ModelPath = TEXT("D:/Models/gemma.bin");
FLiteRtLmUnrealApi::LoadModel(Config);

// 2. 发起对话 (Context-Aware)
FLiteRtLmUnrealApi::ChatWithPrompt(
    this, 
    TEXT("你好，请介绍一下你自己。"),
    TEXT("你是一个乐于助人的游戏向导。"),
    FLiteRtLmChunkCallback::CreateLambda([](const FString& Chunk) {
        // 实时打字机效果
        UE_LOG(LogTemp, Warning, TEXT("AI: %s"), *Chunk);
    }),
    FLiteRtLmDoneCallback::CreateLambda([](const FLiteRtLmResult& Result) {
        UE_LOG(LogTemp, Display, TEXT("推理耗时: %.2f ms"), Result.TimeMs);
    })
);
