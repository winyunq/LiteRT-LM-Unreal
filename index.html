ABI Firewall: Zero-Deployment Integration

We establish a physically isolated ABI firewall through litert_lm_wrapper.dll. This means that when integrating 70+ low-level dependencies (Bazel, Rust, TensorFlow, WebGPU), you don't need to install any complex build environments on the host machine. The plugin is plug-and-play.
